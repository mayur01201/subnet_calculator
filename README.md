# Subenet calculator

This program is written for Python2 version only. Will not work on Python3 or above.
```
This program calculate follwing information:

1) Wild card mask
2) No. of host per subnet
3) Network address
4) Broadcast address
5) Generate random IP address
```
## Usage

```
python main.py
```
